About Galardex

Galardex is a tool to assist players in their quest to catch em' all in the Pokémon Sword and Shield game. Galardex covers the Sword and Shield base game, and can be used to help you navigate the expansive Galar Region, assemble your team, and become the champion.

How It Works

Filter

Using the panel on the left side of the screen you are able to filter through the various Pokémon found in the Galar Region.

The Name filter can be used to search for a specific Pokémon by name.

The Type 1 and Type 2 selectors allow you to filter by the Pokémon's primary and secondary types.

The Location filter will let you filter to Pokémon that spawn in a specific area of the Galar region. When using this filter the map will only show the dots (spawn locations) of Pokémon within the region, despite the fact that the Pokémon that spawn there may also spawn in other regions.

The Generation filter will allow you to filter by the generation in which that Pokémon was introduced.

The HP slider lets you filter the list to only show Pokémon whose HP (health) is within the given bounds.

The Spawn Chance slider lets you filter to only show Pokémon that have a chance of spawning within the bounds. Similarly to the location filter, the map will only show the specific dots that spawn within the restrictions even if that Pokémon spawns elsewhere on the map at a different chance.

Search

The middle of the visualization is a large table that lists out all of the Pokémon that meet the given filter requirements.

Clicking on one of the rows will change the left section from the filter panel to an in-depth Pokémon details panel where you can find out about the stats of the Pokémon.

Additionally, selecting a Pokémon will filter the map panel to only show the dots that correspond with that Pokémon.

Explore

The right panel is a map of the Galar Region with the dots representing a Pokémon spawn location.

Dots are colored according to the primary type of the Pokémon, and hovering on a dot allows you to see more information about the Pokémon it represents.

Similarly to the table, clicking on a dot will select the Pokémon, open up the details panel, and filter the map dots.

About The Project

Galardex is a project created by Maddie Cox, Sabrina Whitney, Mia Quintanilla, and Lilah Bevins for CSCI 3395: Data Visualization at Trinity University. The GitHub repo for this program can be found here.

Tech Stack

HTML, CSS, Javascript

HTML, CSS, and Javascript make up the core components of our visualization.

D3

We used d3.js to parse our data files and render the map visualization.

Earcut

We used the earcut.js library to generate random points within the locations of the Galar region so that we could accurately display the physical locations of Pokémon spawn areas.

noUiSlider

We used the noUiSlider javascript library to create the HP and Spawn Chance sliders for the filter panel.

Data

Pokémon All Status Data by Takamasa Kato on Kaggle

This dataset includes statistics about all 8 generations of Pokémon from across all of the Pokémon properties.

Pokémon Sword and Shield Pokémon Locations by rankedboost

This dataset contains information about the spawn chances of the Pokémon at each of the locations within the Galar Region. This data did not come preformatted in a CSV file, so we manually created one for this project that can be found here. For the sake of time we weren't able to include the information about how weather effects spawn chances, but that is information that we could use in the future.

Galar Pokedex

This is a file that we generated using a script in order to pair down the Pokémon All Status Data to only the Pokémon that spawn in the Galar Region of the Sword and Shield game.

Galar Images

We weren't able to find a data set that mapped Pokémon names to images, so we manually created the CSV file.

Galar Locations Map Data

This is a dataset that our team created with the (x, y) coordinates that describe the path around each of the locations on the Galar map. This data was used to create the map part of our visualization.

Interesting Findings

Using this data visualization we were able to discover specific regions within the Pokémon world where the individual pokemon were spawning more and less frequently. When looking at the map, you can easily notice the different densities of Pokémon in each respective region. Using the Filter section, you can easily get a closer look at a certain location or a specific type of Pokémon.

We found that some of the locations with the highest pokemon spawning density are Dusty Bowl and Lake of Outrage. On the other hand, some of the least populated pokemon areas are Hilbury and Motostoke.

We hope this data visualization brings you as much entertainment as it does for us. Please, explore and further discover the world of Pokémon!
